# Project Summary
The **Style Bank** project is a fashion platform designed to connect creators, sellers, and users through outfit inspiration and shopping. It provides a comprehensive outfit management application that allows users to organize clothing items, manage outfits, and participate in a resale marketplace. The platform enhances user engagement through a points system and social media integration, fostering a vibrant community.

# Project Module Description
- **Outfit Management**: Create and manage outfits with images and product details.
- **Closet Management**: Track clothing items, including purchase details and wear history.
- **Resale Features**: Evaluate resale prices based on wear and depreciation.
- **User Statistics**: Insights on spending, points earned, and clothing usage.
- **Instagram Integration**: Link Instagram posts to outfits for better sharing.
- **Authentication & User Management**: Sign up, log in, and manage profiles.
- **Outfit Feed**: View and interact with outfits created by users.
- **Outfit Bank**: Create and submit new outfit cards for sharing.
- **Sellers and Creators Pages**: Explore and filter sellers and creators based on various metrics.

# File Description Inventory
- **README.md**: Overview and setup instructions.
- **package.json**: Lists dependencies and scripts.
- **src/App.tsx**: Main application component that sets up routing and protected routes.
- **src/pages/**: Contains pages like Index, Closet, OutfitBank, Resale, OutfitDetail, MyPage, Sellers, Creators, and Compare.
- **src/components/**: Reusable UI components.
- **src/lib/**: Utility functions for calculations and local storage, including new database functionalities.
- **src/types/**: TypeScript interfaces for application objects.
- **src/hooks/**: Custom React hooks for managing state and logic.
- **src/contexts/**: Context providers for authentication and state management.
- **src/pages/Index.tsx**: UI for the main user dashboard, displaying user stats and quick actions.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for faster development.
- **Shadcn-ui**: UI component library.
- **Supabase**: Backend as a service for database and authentication.
- **localStorage**: For persistent storage of user data.

# Usage
1. Install dependencies: 
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run dev
   ```
