# 셀러/크리에이터 비교 기능 구현

## 구현할 파일 목록

1. **src/pages/Index.tsx** - 홈 대시보드에 3개의 탐색 버튼 추가
2. **src/pages/Sellers.tsx** - 셀러 비교 페이지 (프로필, 상품 수, 평균 가격, 평점)
3. **src/pages/Creators.tsx** - 크리에이터 비교 페이지 (Outfit 수, 좋아요, 수익, 팔로우)
4. **src/pages/Compare.tsx** - 상품 비교 페이지 (가격/리뷰/인기도 시각적 비교)
5. **src/lib/database.ts** - 샘플 셀러/크리에이터 데이터 추가, 팔로우 기능
6. **src/App.tsx** - 새 라우트 추가

## 디자인 컨셉
- 20~30대 타겟: 밝고 트렌디한 그라데이션, 인스타그램 스타일
- 비교 기능: 카드 레이아웃, 필터/정렬, 시각적 차트
- 인터랙션: 좋아요, 팔로우, 북마크 기능
